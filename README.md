# Grocery Billing System

A beginner Python project that simulates a grocery store billing system.

## Features

-  Add grocery items to the cart
-  Check whether an item is available in the inventory
-  Calculate the total bill using different prices for different products
-  Replace unavailable items
-  Gold membership discount
-  Membership ID verification
-  Display the final bill and total number of items purchased

## Concepts Used

- Python Variables
- Lists
- Dictionaries
- Functions
- While Loops
- If-Else Statements
- User Input
- Global Variables

## Technologies

- Python 3
- Google Colab
